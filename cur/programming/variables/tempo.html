But it doesn't have to be so difficult to change the speed of a song! We can keep track of a variable that we will call "tempo" that will change how long 1 beat is.

    In the sounds tab find the tempo block and select the check box next to "tempo" so that you can see the value of the variable "tempo" on the stage.

    Now double-click on the tempo blocks and see how the variable tempo changes!

    Make a song and change the tempo..
