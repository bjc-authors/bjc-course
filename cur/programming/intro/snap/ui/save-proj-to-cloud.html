2. To Save Your Snap! Project to the Snap! Cloud or Browser

    The point of saving your project to the Snap! cloud or browser is so that you won't have a lot of Snap! projects sitting around on your desktop.

    A. Saving to the Snap! Cloud

    Click on the file icon button on the upper left corner of the Snap! window, and then click on the "Save" option.

    (---insert file icon button image with save option highlighted on drop down menu image---)

    In the Save Project window, make sure the cloud button is highlighted in blue on the left hand side of the window. Name your project in the space for you type in a title. Afterwards, click on the save button.

    (---insert Save Project window image with cloud button highlighted in blue (leave as is basically) and the words "Enter Title Here" in the title box---)

    You should see the following blurbs appear: (1) Saving Project to the Cloud and (2) Saved.

    Yay! Your project is now saved!
