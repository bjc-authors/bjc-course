2. To Save Your Snap! Project to the Snap! Cloud or Browser

    B. Saving to your Web Browser

    Follow the instructions in Part A to get to the Save Project window.

    In the Save Project window, make sure the browser button is highlighted in blue on the left hand side of the window. Name your project in the space for you type in a title. Afterwards, click on the save button.

    Afterwards, click on the save button.

    Doing this will change your browser's url to reflect a url that links directly to your project. Your browser will save a copy of this so whenever you open up your browser, you can access you file.
