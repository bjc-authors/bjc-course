Exercise

    With all that you have learned about lists in this lab and in the previous lab, you are ready to make a high-score list that stores the names and scores of players of a game. The list keeps track of the names of the players and the scores (in order) as shown below. The goal is to write a block that allows you to add a new player to your high-score list. You may find this block useful in later projects.


    To complete this task, you will make the following blocks that will help you build an abstraction layer.
    • Get name from/Get score from: Instead of always manually accessing the score or the name from a list element, you can write blocks that take a list and an index as inputs and report either the name or score of a player.
    • Add new score: This block should add the name provided (Colleen) along with the new score (52) that is also provided as input. The new score entry should appear in the correct order.

    Test Cases

    As we discussed in previous labs, it is important to test your script on different kinds of input: input that is "regular" or "normal", and input that exercises the "edge cases" of the script. For example, for the Add New Score block, we recommend the following test cases:
    • Adding a new score to an empty list.
    • Adding a new score that should be placed in the middle of a non-empty list.
    • Adding a new score that should become the highest score in a non-empty list.
    • Adding a new score that should become the lowest score in a non-empty list.
    • Adding a new score that is the same as another score in a non-empty list.
    Your homework should include evidence of these tests; you may want to make a different list for each test case. To each test case, attach a comment that describes what is being tested. To make a comment, right-click anywhere on the Scripts section and select add comment.
    BYOB will then provide a pale yellow box where you can add a comment.
    Once a comment is attached to a block, the comment will follow the block wherever the block is dragged.
