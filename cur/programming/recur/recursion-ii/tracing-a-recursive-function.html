What really happens when we call the function below with the argument 3?


    The flow chart below attempts to show what actually happens! The arrows are numbered to show the sequence of events. We didn't show the sequence of events within each function because we assume you feel comfortable tracing the individual functions.
    • The functions have the argument "x" replaced by the actual value.
    • The black arrows show "delegation" to another COPY of the function (i.e. a recursive call)
    • The white arrows show the return of control to the calling function. If the caller had any lines after the recursive call, those would be executed. Later we will work with command blocks, and the white arrow will indicate the recursive call returning a value.
