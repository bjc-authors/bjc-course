Abstraction

    The picture below is a simple visualization of a brick wall that can be drawn using Snap!

    In this activity you will be drawing this same brick wall. Remember to use functional decomposition as much as you can so that your final block is easy to understand. The following hints might be helpful:

    1. You can use a thick red pen to draw a single brick.

    2. There are two types of rows both composed of bricks.

    3. Not all of the bricks are the same size.
