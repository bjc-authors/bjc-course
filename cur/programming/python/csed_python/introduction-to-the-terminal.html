What's a Terminal?

    You're used to navigating around your computer by clicking on folder icons and using menus to create new files and folders. We can do all of this from the terminal as well, by typing out commands instead of using the mouse to click on things.

    Opening a Terminal

    To open a terminal, first click on the swirly icon on the top left of the lab computer screen, shown in the picture below.

    Once you've clicked on it, a window should pop up that says "Search your computer and online sources" as shown below.

    In that box, type in the word terminal, and a list of applications should pop up. This list should include terminal, as shown below.

    Click on terminal, and you should see a window appear that looks similar to the window below.

    The little bit of text that looks similar to s273-2 [1] ~ # is called the prompt. When you type commands, they appear to the right of the prompt.

    Let's learn how to navigate around. Right now, you're in the home directory, also called ~. A directory is just another name for a folder (like the Documents folder).

    ls: listing contents of a directory

    To see what's in the current directory, we can use the command ls. The ls command lists the contents of our current directory. To use a command, type it in the terminal, then press 'enter'. Try it out. You should see something like this: 

    
s273-2 [1] ~ # ls
Desktop  Documents  Downloads  Music  Pictures  Public  Templates  Videos

    cd: navigating to another directory

    We can see that the home directory has some other directories in it, such as Documents and Desktop. If we want to go into one of those directories, we can use the command cd, followed by that directory's name. Try it out by using the command cd Documents. 

    
s273-2 [2] ~ # cd Documents
s273-2 [3] ~/Documents #

    Now you're in the Documents directory. We can use ls again to see the contents of Documents. If you're the first person using the computer of the day, you should notice that nothing appears, which means that the directory is empty. If anything appears, for example hello.py, lab1.py, or anything else, that's something another student created earlier in the day.

    If you see an earlier students work, mercilessly destroy it using the command rm *.py and answer yes when prompted. 

    
s271-22 [2] ~/Documents # ls
hello.py
s271-22 [3] ~/Documents # rm *.py
rm: remove regular file ‘hello.py’? yes

    You can also use the terminal to do almost anything, such as moving, copying, and removing files and directories. If you would like an explanation of some other useful terminal commands check out this article.
