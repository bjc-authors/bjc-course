Running a Python File

    To write a python program, we first need to open a text editor. A text editor is like Microsoft Word or Google Docs, but only supports plain old boring text. We'll be using "Sublime Text", which is installed on the lab machines. Click on the swirly icon in the top left again. You'll probably see the word "terminal" still there. Delete it, then type in "sublime" and a "Sublime Text 2" icon should appear. Click on this icon and you should see the Sublime Text 2 editor open up, as shown below.

    We'll start by saving our blank file and giving it a name. To do this, press control-S, and a window should appear that looks like the picture below.

    Press delete to erase "untitled", and type "hello.py" instead. Then click the "Documents" folder on the left side. Once you've done both of these things, your screen should look like this:

    If you're getting a little lost, make sure to ask one of the lab assistants! Once you feel like you've reached this step, click "Save" in the bottom right hand corner. Once you've done, this, you've now saved your first program.

    Of course, a blank program is pretty boring, so in the Sublime Text window, type exactly print("Hello World"). Once you've done this, your sublime window should look as follows:

    It's very important that the text should be showing up in color. If it's showing up still all white, ask a lab assistant for help. Now press control-S to save the changes to your program, which will make the little dot in the title bar go away, as shown below. If you look carefully you can see there is no longer a little tiny dot to the right of "hello.py" in the title bar.

    Now finally, it's time to run your program from the terminal. Go back to your terminal window and use the ls command to verify that hello.py is there! When you type ls, you should see something like the following. Your terminal will be black and white, with no color. That's normal. 

    
s273-2 [3] ~/Documents # ls
hello.py

    To run your program, just type python3 hello.py. You should see the words Hello World printed to your terminal. It should look a little something like this: 

    
s273-2 [4] ~/Documents # python3 hello.py
Hello World

    You've just run your file with Python!
