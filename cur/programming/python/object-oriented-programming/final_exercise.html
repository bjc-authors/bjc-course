Okay, now that you've learned the basics of object-oriented programming in Python, it is time for you to create your very own class! If you look below your Textbook class, you'll see the class UCBMFET. Your job is to fill it out!

    • If you're not sure what UCBMFET is, it stands for UC Berkeley Memes for Edgy Teens. It's a Facebook page where both Berkeley and non-Berkeley students post funny jokes and memes about Berkeley. It's currently the largest college meme page that exists with roughly 173,532 members! Since there are so many people, moderators often regulate the page make sure the memes are original and everyone is an actual member of the group.

    To-Do

    1. The following class attributes:
      1. members, initialized to 173532
      2. topic, initialized to 'memes'
      3. posts, intialized to an empty dictionary
    2. A constructor method which does the following:
      1. Sets an instance attribute name to the input parameter of the same name
      2. Sets two instance attributes activity and posts both to 0.
      3. Sets an instance attribute member to True
      4. Increases the class attribute member by 1
    3. Completion of the tag_your_friend_in_meme so that it does the following:
      1. If the input parameter friend is not an instance of the UCBMFET class, returns string: 'You cannot tag someone in a meme if they are not a member of UCBMFET.' (Hint: You can use isinstance(object, class) to check if an object is an instance of a class)
      2. Otherwise it increases your activity (not your friend's) by 1 and returns the string: '@ [insert friend's name here] us in CS10!'
    4. Completion of the method post_in_UCBMFET so that it does the following:
      1. If the parameter of title_of_post already exists as a key inside the class attribute posts, returns the string, 'You have been banned for reposting a meme.'
      2. Otherwise, it adds the title_of_post as a key inside the class attribute posts, with the value initialized to 0 (corresponding to the number of likes so far). It increases your activity and posts each by 1. It also returns the string 'Your total activity on UCMBFET is [insert your activity value here], and your total posts to UCBMFET is now [insert your posts value here].'
    5. Completion of the method like_a_post_in_UCBMFET so that it does the following:
      1. Increases your activity by 1
      2. Increases the value of the key which corresponds to the post's title in the class attribute posts by 1.
      3. Returns the string 'Your total activity on UCBMFET is [insert your activity value here, and the total number of likes on the post
        [insert title_of_post] is [insert number of likes].'

    Congratulations! You have just built your very own class!
